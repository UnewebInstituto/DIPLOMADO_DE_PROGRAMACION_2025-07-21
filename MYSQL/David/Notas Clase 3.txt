Normalizacion de base de datos:

La normalización de bases de datos es el proceso de organizar los datos en una base de datos para reducir la redundancia y mejorar la integridad de los datos. Se aplica mediante un conjunto de reglas conocidas como formas normales, que ayudan a eliminar datos duplicados y a asegurar que la información esté almacenada de manera lógica y consistente.
